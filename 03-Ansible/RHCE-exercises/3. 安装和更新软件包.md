## 3、安装软件包
安装软件包
创建⼀个名为 /home/greg/ansible/packages.yml 的 playbook :
- [ ] 将 php 和 mariadb 软件包安装到 dev 、 test和 prod 主机组中的主机上
- [ ] 将 RPM Development Tools 软件包组安装到 dev 主机组中的主机上
- [ ] 将 dev 主机组中主机上的 所有软件包更新为最新版本

### 实操过程

#### 查找文档
- `ansible-doc yum`
- 这道题基本上就照抄文档里面的例子就可以了

#### 编写剧本
- `vim packages.yml`
```yml
- name: install1
  hosts: dev,test,prod
  tasks:
    - name: install php,mariadb
      yum:
        name: "{{ packages }}"
        state: present
      vars:
        packages:
        - php
        - mariadb
- name: install2
  hosts: dev
  tasks:
    - name: install package group
      yum:
        name: "@RPM Development Tools"
        state: present
    - name: update
      yum:
        name: '*'
        state: latest
```
- 有两个注意的点：
- 变量名外面需要双引号
- 更新的时候的*外侧要用单引号，因为用双引号的话会解析成任意字符

#### 验证
- `ansible dev,test,prod -a 'rpm -q php mariadb'`
- 查看是否安装成功
- `ansible dev -a 'yum grouplist'`
- 看已经安装的软件包，里面应该有下面这样的信息：
```
Installed Groups:
 RPM Development Tools
```
- `ansible dev -a 'yum update'`
- 查看是否不需要更新了，如果显示"nothing to do"就对了
