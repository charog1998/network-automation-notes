## 7、创建和使用角色 
创建和使用角色
根据下列要求，在 /home/greg/ansible/roles 中创建名为 apache 的⻆⾊：
- [ ] 安装 httpd 软件包，设为在 系统启动时启⽤ 并 启动
- [ ] 启⽤防⽕墙 并运⾏，并使⽤允许访问 Web 服务器的规则
- [ ] 创建模板⽂件 index.html.j2，⽤于创建具有以下输出的⽂件 /var/www/html/index.html ：
> Welcome to HOSTNAME on IPADDRESS
- [ ] 其中，HOSTNAME 是受管节点的 完全限定域名 ， IPADDRESS 则是受管节点的 IP 地址。
- [ ] 创建⼀个名为 /home/greg/ansible/apache.yml 的 playbook：
- [ ] 该 play 在 webservers 主机组中的主机上运⾏并将使⽤ apache ⻆⾊

### 实操过程

#### 初始化角色
- `ansible-galaxy role init --init-path /home/greg/ansible/roles apache`

#### 编写task
- `vim roles/apache/tasks/main.yml`
```yml
---
- name: Install Apache
  # 安装httpd
  ansible.builtin.yum:
    name: httpd
    state: latest
  # 打开httpd
- name: Start httpd
  ansible.builtin.systemd:
    name: httpd
    state: started
    enabled: yes
  # 打开firewalld
- name: Start firewalld
  ansible.builtin.systemd:
    name: firewalld
    state: started
    enabled: yes
  # 在防火墙里打开http服务
- name: Start apache
  ansible.posix.firewalld:
  # 注意防火墙中服务名不是httpd
    service: http
    permanent: yes # 是否在防火墙重启后仍生效
    state: enabled
    immediate: yes # 是否立即生效
  # 利用template在目标主机创建文件
- ansible.builtin.template:
    src: index.html.j2
    dest: /var/www/html/index.html
```
- `ansible.posix.firewalld`这个包的文档，可以用`ansible-navigator collections`打开之后用`:编号`选择对应的镜像来查看，里面也有例子
#### 编写j2模板
- `vim roles/apache/templates/index.html.j2`
```j2
Welcome to {{ ansible_nodename }} on {{ ansible_default_ipv4.address }}
```

#### 编写剧本
- `vim /home/greg/ansible/apache.yml`
```yml
---
- name: webservers role
  hosts: webservers
  roles:
  - apache
```

#### 运行
- `ansible-navigator run apache.yml -m stdout`
