## 17、配置 cron 作业
配置 cron 作业
创建⼀个名为 /home/greg/ansible/cron.yml 的 playbook :
- [ ] 该 playbook 在 test 主机组中的受管节点上运⾏
- [ ] 配置 cron 作业，该作业每隔 2 分钟 运⾏并执⾏以下命令：logger "EX200 in progress" ，以⽤户 natasha cat身份运⾏

### 实操过程

#### 编写
- `vim cron.yml`
```yml
---
- name: create cron task
  hosts: test
  tasks:
    - name: create task
      ansible.builtin.cron:
        name: "cron task"
        job: 'logger "EX200 in progress"'
        minute: '*/2'
        user: natasha
```

#### 运行
- `ansible-navigator run cron.yml -m stdout`

#### 验证
- `ansible test -a 'crontab -l -u natasha'`
- 能看到natasha用户下的定时任务
- `ansible test -m shell -a 'cat /var/log/messages |grep "EX200 in progress"'`
- 能看到确实每隔两分钟输出了一次
