## RFC 阅读器

基于 [Textual](https://github.com/Textualize/textual) 的终端交互式 RFC 阅读器，自动按换页符分页，支持搜索高亮、目录导航、书签和文件浏览器。

### 安装依赖

```bash
pip install textual
```

### 用法

```bash
# 文件浏览器模式（默认）—— 光标选择文件
python rfcview.py

# 直接打开指定文件
python rfcview.py 01-转发与控制/rfc4271-bgp4.txt

# 从第 50 页开始阅读
python rfcview.py rfc4271-bgp4.txt -p 50
```

### 按键说明

**文件浏览器**

| 按键 | 功能 |
|------|------|
| `↑` `↓` / `j` `k` | 上下移动光标 |
| `Enter` / 双击 | 打开文件 / 进入目录 |
| `←` / `Backspace` | 返回上级目录 |
| `s` | 切换排序模式（按名称 / 按类型 / 按时间） |
| `h` `?` | 帮助 |
| `q` / `Esc` | 退出 |

**RFC 阅读器**

| 按键 | 功能 |
|------|------|
| `n` `→` `Space` `j` | 下一页 |
| `p` `←` `Backspace` `k` | 上一页 |
| `PgDn` / `PgUp` | 前进/后退 5 页 |
| `Home` / `End` | 首页 / 末页 |
| `g` | 跳转到指定页码 |
| `t` | 目录导航（按 RFC 章节跳转） |
| `/` | 搜索关键词（黄色高亮） |
| `m` | 标记书签（记录当前位置） |
| `'` | 跳回书签 |
| `l` | 切换行号显示 |
| `h` `?` | 帮助 |
| `q` / `Esc` | 退出（自动回到文件浏览器） |

> 弹窗（跳转、搜索、目录、帮助）中均可用 `q` 或 `Esc` 关闭。

### 功能特性

- **自动分页**：按 RFC 原文换页符 `\f` 切割页面，去除页眉页脚
- **搜索高亮**：关键词统一以黄色反白显示，跨页自动搜索
- **目录导航**：解析 RFC 目录页，`t` 键弹出章节列表，`Enter` 跳转
- **临时书签**：`m` 标记当前页和滚动位置，`'` 一键跳回，适合对照阅读
- **行号切换**：`l` 键开关行号显示，淡蓝色与正文区分
- **文件排序**：文件浏览器按 `s` 在名称、类型、时间三种排序间循环，时间按最新在前排列
- **时间列**：文件列表中自动右对齐显示修改时间
- **双击打开**：双击文件同样触发打开操作
- **目录记忆**：从阅读器退回文件浏览器时自动回到刚才的目录
