## 计费方式

共享带宽支持包年包月和按量计费两种付费模式。

-   **包年包月**：是一种需要先付费才能使用资源的付费模式。包年包月共享带宽实例的特点如下：
    
    -   在创建共享带宽实例时付费。
        
    -   在当前合同期内，共享带宽实例只支持升配，不支持降配和释放。
        
        共享带宽实例升配操作，请参见[包年包月实例变配](https://help.aliyun.com/zh/internet-shared-bandwidth/product-overview/change-configurations-of-subscription-instances#task-pmk-tcy-b2b)。如需对共享带宽实例降配，可通过续费变配实现，降配后的带宽峰值将在续费周期内生效。具体操作，请参见[续费变配](https://help.aliyun.com/zh/internet-shared-bandwidth/product-overview/change-configurations-of-subscription-instances#section-xq5-72d-431)。
        
-   **按量计费**：是一种先使用后付费的付费模式。按量计费共享带宽实例的特点如下：
    
    -   按量计费共享带宽实例支持升配和降配，还支持转换计费方式。具体操作，请参见[按量计费实例变配](https://help.aliyun.com/zh/internet-shared-bandwidth/product-overview/change-configurations-of-pay-as-you-go-internet-shared-bandwidth-instances#task-827099)。
        
    -   按量计费（按带宽）支持转换为包年包月。具体操作，请参见[转换计费方式](https://help.aliyun.com/zh/internet-shared-bandwidth/product-overview/switch-billing-method#task-827266)。
        

不同付费模式下支持不同的公网计费方式。各计费方式适用场景不同，您可以根据业务场景选择合适的计费方式。

| **付费模式** | **公网计费方式** | **适用场景** |
| 按量计费 | 按带宽计费 | - 流量峰值比较稳定。 - 资源使用有临时性和突发性。 |
| 按增强型95计费 | - 业务流量峰值较大。 - 流量峰值波动较大，且波动比较频繁。 |
| 按主流量计费 | - 流量峰值波动较大。例如游戏、视频等行业。 - 资源使用有临时性和突发性。 如您无法预估流量使用情况时，建议选择按使用流量计费。 |
| 包年包月 | 按固定带宽计费 | - 可预估资源使用周期。 - 流量峰值比较稳定。 - 需要长期使用资源（价格比按量付费更优惠）。 |

各计费方式在带宽稳定性方面存在差异，选择计费方式时请结合业务对带宽稳定性的要求综合考虑。

-   按带宽计费（包括包年包月按固定带宽和按量计费按带宽）：有带宽保障承诺，实际可使用的带宽峰值稳定，适合对带宽稳定性要求较高的业务场景。
    
-   按流量计费（按主流量计费）：不承诺带宽峰值，实际可使用的带宽可能随网络负载波动。按实际产生的主流量（入方向和出方向流量取较大值）计费，费用与设置的带宽峰值无关。适合对带宽稳定性要求不高但希望按实际使用量付费的场景。
    
-   按增强型95计费：有保底带宽承诺，保底带宽范围内带宽稳定可靠，超出保底带宽部分按去峰后的月平均峰值结算。适合流量峰值波动较大但需要一定带宽保障的场景。
    

如果您的业务对带宽稳定性要求较高，建议选择包年包月按固定带宽计费或按量计费按带宽计费。

## **更多信息**

### **欠费说明**

#### **包年包月实例欠费说明**

包年包月实例欠费后：

-   包年包月到期后将停止服务，并将为您保留7天。
    
-   如果您在7天内在控制台或[续费管理中心](https://renew.console.aliyun.com/center?spm=a2c4g.11186623.2.4.mamRKK#/renew/cbwp-pre?_k=7bynmu)完成续费，实例将恢复正常使用状态。
    
-   如果7天内未完成续费或续费不成功，7天后实例将被释放。共享带宽内的弹性公网IP将自动退出共享带宽并恢复为加入之前的带宽峰值和计费方式。
    

#### **按量计费实例欠费说明**

按量计费实例欠费后的服务状态变化如下：

-   账号当前的可用余额（含现金、相关代金券、优惠券或预授权等）不能结清已出的账单时，系统即判定账号欠费。
    
-   欠费后如果在延停权益额度内，您的服务将不会受到停服影响。
    
    **说明**
    
    阿里云提供延期免停权益，即当按量付费的资源发生欠费后，提供一定额度或时长继续使用云服务的权益，延停期间正常计费。具体使用说明和规则，请参见[延期免停权益](https://help.aliyun.com/zh/user-center/exemption-of-suspension-after-delinquency)。
    
-   欠费后如果超出了延停权益额度，共享带宽将停止工作，并为您保留7天。
    
-   如果您在7天内缴清欠费账单，实例将恢复正常使用状态。
    
-   如果7天内未缴清欠费账单，7天后实例将被释放。共享带宽内的弹性公网IP将自动退出共享带宽并恢复为加入之前的带宽峰值和计费方式。
