阶段一：全局认知

先看森林，再看树木

1 网络服务选型指南
理解阿里云网络产品全景，每个产品解决什么问题		 文档索引 → 选型与规划设计 → 网络服务选型指南 2h
2 云网络产品基础方案
产品之间的协作关系、典型搭配模式		 文档索引 → 选型与规划设计 → 产品基础方案 2h
📖

阶段二：按权重逐个攻克产品

权重越高越先学，关联产品打包学
🎯 目标：每个产品能说清「核心功能 + 适用场景 + 计费模式 + 常见配置参数」
# 产品 考纲权重 学习重点 题库
1 智能接入网关 SAG 15% 三种形态（硬件/软件/APP）、SD-WAN 组网、混合云接入场景、带宽配置 36题
2 负载均衡 SLB 12% ALB(7层) / CLB(经典) / NLB(4层高性能) / GWLB(网关) 四者区别、监听规则、健康检查、会话保持、SSL卸载 66题
3 弹性公网 IP + NAT 网关 打包学 15% EIP 绑定/解绑/共享带宽、SNAT（共享上网）/ DNAT（端口映射）、公网出口方案组合。实际场景总是搭配使用，放一起学效率最高 39+22题
4 云企业网 CEN 10% 转发路由器 TR、跨地域带宽包、路由策略、网络实例连接、多VPC互联与VPC对等连接的区别 52题
5 全球加速 GA 10% 加速IP / 终端节点组 / 监听 / Endpoint、智能路由就近接入、与 CDN 的定位区别 36题
6 专有网络 VPC 8% CIDR 规划、交换机 / 安全组 / 网络 ACL、VPC 对等连接、PrivateLink 私网连接 72题
7 VPN 网关 5% IPsec-VPN（站点到站点）vs SSL-VPN（点到站点）、BGP 配置、IKE 协商 19题
8 高速通道（专线） — 物理专线、共享专线、与 VPN 的对比选型（高带宽低延迟 vs 低成本快速部署） 12题
9 安全四件套 打包学 ~5% DDoS 防护 / WAF / 云防火墙 / 访问控制 RAM，重点掌握与网络产品（SLB/CDN/VPC安全组）的联动部署方式 —
10 CDN + DNS 打包学 5% CDN 缓存策略 / 回源配置 / 加速域名；云解析 DNS 智能解析 / GTM 全局流量管理；PrivateZone 内网解析 13+17题
11 共享带宽 + 共享流量包 打包学 2% 计费模式差异（包年包月 vs 按量）、适用场景、与 EIP 的配合使用 20题
💡 学习闭环
读产品概述 → 刷该标签题库 → 读用户指南关键章节 → 标记错题 → 回头重看文档中不理解的部分

阶段三：场景综合分析

决定 70 → 85+ 的关键
🎯 目标：拿到一个业务需求，能画出网络架构图，并对每个节点说出选型理由
#学习内容对应文档建议时间
1 云网络白皮书 文档索引 → 选型与规划设计 → 云网络白皮书 3h
2 架构设计指南 文档索引 → 选型与规划设计 → 架构设计指南 4h
3 场景化方案 文档索引 → 选型与规划设计 → 场景化方案 5h

重点吃透四类经典场景

🏢 企业上云

IDC → 阿里云，SAG vs VPN vs 专线怎么选？成本、带宽、延迟如何权衡？

🌐 多地域组网

多 VPC 跨地域互联，CEN vs VPC 对等连接怎么选？路由如何规划？

🚀 全球加速

海外用户访问国内服务，GA + CDN 怎么搭配？加速 IP 与源站如何布局？

📡 公网出口设计

EIP + NAT 网关 + 共享带宽 + 安全产品，怎么组合最省钱又安全？

🔍 检验方式
题库"场景分析"标签（24题）+ 不设标签随机混合刷题，检验能否在跨产品场景中做出正确判断
🚀

阶段四：冲刺刷题

326题全量演练
🎯 目标：随机抽100题限时120分钟稳定 80+ 分，错题清零
  1. 分标签清仓 每个标签用 qz --tag <id> 逐标签刷一遍，标记错题，回顾对应文档
  2. 随机全量刷 不设标签 qz，326题随机打乱，检验跨产品边界的综合判断力
  3. 搜索补漏 用 REPL 的 sr 命令搜薄弱关键词（BGP、健康检查、SNAT、回源、CNAME、SSL卸载 等），逐条吃透
  4. 全真模拟 随机抽 100 题，限时 120 分钟，闭卷。目标 80 分以上。至少做 2 次，直到稳定达标
