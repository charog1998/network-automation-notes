阿里云官方提供的面向公网的权威DNS，您可将域名添加进来并配置解析，即可在公网通过域名访问到对应服务。

## **概述**

**公网权威解析**服务，是基于阿里云全自研高性能DNS软件打造的安全、快速、稳定、可靠的互联网权威解析服务，可将易于识别的域名转换为网络IP地址，从而让用户定位到相应的网站或应用服务器。企业在购买域名用于建站时，都会用到**公网权威解析**服务，企业管理员可在**公网权威解析**服务中配置域名解析记录，即配置业务域名到目标服务IP地址的解析关系。当众多网络用户访问企业域名时，用户终端使用本地LocalDNS从**公网权威解析**服务获取到域名解析目标地址，最终连接到目标地址服务器访问网络服务内容。

## **应用场景**

| **应用场景** | **场景描述** |
| --- | --- |
| 网站建设 | 客户可以通过应用A记录，将网站域名指向网站的服务器地址，来实现用户可以打开网站的效果。 |
| 电子邮箱 | 客户可以通过应用MX记录，根据邮箱服务商提供的配置来设置解析记录，从而达到电子邮箱的收发邮件功能。 |
| 高访问量的业务 | 当多台服务器都服务于同一个业务时，可利用加权轮询解析机制，实现将流量分摊到每台服务器上，以此分散业务压力。 |
| 用户跨网/跨地域访问的场景 | 当用户分散于不同的运营商或者地域，通过智能解析配置，可根据用户不同的地理位置或网络环境来智能返回解析结果。 |
| CDN加速业务 | 客户可以通过应用CNAME记录指向CDN服务商提供的别名，从而最终实现提高用户访问网站的响应速度或者下载速度。 |

## **系统架构**

**公网权威解析**是由管控层和解析层两部分组成：

-   管控层：管控层通过控制台和OpenAPI对外提供服务，主要实现了域名解析数据、配置数据、日志数据等的增删改查功能和存储功能。管控层位于中国内地-华北3（张家口）、中国内地-华东1（杭州）。
    
-   解析层：解析层通过部署在全球范围内的解析服务器集群对外提供服务，解析层接收来自管控层分发的域名解析记录数据，主要实现了对域名解析记录数据的查询请求进行应答的功能。解析层在全球主要大洲和地区均有节点覆盖。
