## **在阿里云购买了域名，想进行网站解析配置**

1.  如果域名显示**ClientHold**或**ServerHold**等锁定状态，请参考[查看DNS服务器状态及异常状态处理](https://help.aliyun.com/zh/dns/pubz-view-dns-server-status-and-exception-handling)。
    
2.  提前完成域名持有者的邮箱认证，参考[验证域名持有者邮箱](https://help.aliyun.com/zh/dws/user-guide/verify-the-email-address-of-a-domain-holder)。
    
3.  根据政策要求，解析至中国内地服务器的网站、App等服务，必须完成 [ICP备案](https://help.aliyun.com/zh/icp-filing/basic-icp-service/product-overview/what-is-an-icp-filing#concept-nx4-hql-zdb)才可对外提供服务。
    
4.  参考如下文档配置解析。
    
    -   如果您已经部署了网站，请参考[添加网站解析](https://help.aliyun.com/zh/dns/pubz-add-website-parsing)。
        
    -   如果您购买了邮箱服务，想要为邮箱添加自定义域名，参考[添加邮箱解析](https://help.aliyun.com/zh/dns/pubz-add-mailbox-resolution)。
        
    -   其他更多解析类型，如TXT、NS、URL转发等，参考[添加解析记录](https://help.aliyun.com/zh/dns/pubz-add-parsing-record)。
        
5.  如果您配置解析后仍然无法通过域名访问网站，首先参考[解析生效测试方法](https://help.aliyun.com/zh/dns/pubz-parsing-effectiveness-test-methods)，再参考[解析不生效问题快速排查](https://help.aliyun.com/zh/dns/pubz-quick-troubleshooting-of-problems-not-effective)自助排查解决。

2.  如果您想为域名配置解析，请咨询域名厂商了解域名配置的DNS服务器，并前往进行配置。
    
3.  如果您想让其他厂商注册的域名在阿里云DNS解析，请参考[非阿里云注册域名如何在云解析DNS设置解析](https://help.aliyun.com/zh/dns/how-do-i-resolve-a-non-ali-cloud-registered-domain-name-on-the-alibaba-cloud-dns)。
