## 使用流程

> 本文以按量付费模式为例，为您介绍GA实例的使用流程。包年包月模式下，您需要购买GA实例并为其绑定基础带宽包，才可以配置加速区域和终端节点组等信息。

### 标准型GA实例使用流程

标准型GA实例主要用于四层（TCP和UDP协议）和七层（HTTP和HTTPS协议）网络加速场景。使用流程如下：

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2786993871/CAEQURiBgMCY7rHyqBkiIGRmNWQwNjI2ZmEwYTQwZTZiZmFjYmY3NWZhNWY4MDdl3963382_20230830144006.372.svg)

1.  创建按量付费标准型GA实例。
    
    创建按量付费标准型GA实例时，支持同时配置加速区域、监听和终端节点组。具体操作，请参见[创建标准型实例（按量付费）](https://help.aliyun.com/zh/ga/user-guide/create-and-manage-standard-ga-instances#f5b961f027dh8)。
    
2.  **条件必选：**如果您是通过域名方式加速访问服务，您需要配置DNS将加速域名指向GA实例分配的CNAME地址。
    
    配置CNAME具体步骤，请参见[配置CNAME](https://help.aliyun.com/zh/ga/user-guide/add-a-cname-record-for-a-domain-name)。
    
3.  **条件必选：**如果您的业务配置涉及中国内地的地域与中国香港、中国澳门、中国台湾地域或其他国家和地区之间的访问加速，单击此处查看如何配置跨境服务。
    
    1.  开启跨境服务。
        
        如果您在实例创建时已根据界面提示完成数据跨境合规承诺，则您的实例已开启跨境服务，您可以跳过该步骤。
        
        开启跨境服务后，GA实例的**传输网络质量类型**为精品带宽跨境加速。无需额外配置，可实现中国内地的地域与中国香港、中国澳门、中国台湾地域或其他国家和地区间的访问加速。
        
        1.  在**实例列表**页，找到目标标准型GA实例，单击实例ID。
            
        2.  在实例详情页面，单击**实例信息**页签，在下方**传输网络质量类型**区域，单击**开启跨境服务**。
            
        3.  在**开启跨境服务**对话框，阅读**数据跨境合规承诺**内容并选中**同意以上合规承诺**，然后单击**确定开启**。
            
        4.  在弹出的对话框，单击**确定**。
            
    2.  切换传输网络质量类型。
        
        如果您需要低延迟、高质量的加速效果，您可以将默认的精品带宽跨境加速切换为联通跨境专线。
        
        执行该操作前，请确保已通过联通跨境业务资质申请。
        
        1.  在实例详情页面，单击**实例信息**页签。
            
        2.  在**传输网络质量类型**区域，找到联通跨境专线卡片，单击卡片中的切换联通跨境专线。
            
        3.  如果您是首次启用联通跨境专线加速，在弹出的**切换跨境专线**对话框，选中以下信息，然后单击**切换**。
            
            **警告**
            
            如果您当前的加速配置涉及跨境网络，该切换操作会导致断流1分钟，请谨慎操作。
            
            -   阅读**Cloud Data Transfer服务协议**并选中**我已阅读并同意**，以开通CDT（跨境）服务。
                
            -   阅读切换跨境专线的风险信息并选中**已知晓风险，确认执行**。
                
        4.  如果您非首次启用联通跨境专线加速，在弹出的确认切换为联通专线跨境加速？对话框，选中**已知晓风险，确认执行**。然后单击**切换**。
            
            **警告**
            
            如果您当前的加速配置涉及跨境网络，该切换操作会导致断流1分钟，请谨慎操作。
            
        
        当联通跨境专线卡片显示**启用中**时，表示可使用联通跨境专线加速进行访问加速。
        
    

### 基础型GA实例使用流程

基础型GA实例主要用于三层（IP协议）网络加速场景。使用流程如下：

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2786993871/CAEQURiBgIC09r7yqBkiIDgyNjc3MjRhODU4MzQyYTY5OTZiOGM3MjQwYmFmOTA23963382_20230830144006.372.svg)

1.  创建按量付费且带宽计费方式为**按流量**的基础型GA实例。
    
    具体操作，请参见[创建按量付费基础型实例](https://help.aliyun.com/zh/ga/user-guide/create-and-manage-basic-ga-instances#a577f550881kp)。
    
2.  **条件必选：**如果您的业务配置涉及中国内地的地域与中国香港、中国澳门、中国台湾地域或其他国家和地区之间的访问加速，单击此处查看如何配置跨境服务。
    
    为基础型GA实例开启跨境服务及启用联通跨境专线加速前，需确保已通过联通跨境业务资质申请。
    
    1.  在实例详情页面，单击**实例信息**页签，在下方**付费信息**区域，打开**传输网络质量类型**开关。
        
    2.  在**开启跨境服务**对话框，阅读**数据跨境合规承诺**内容并选中**同意以上合规承诺**，然后单击**确定开启**。
        
        如果您是首次启用联通跨境专线加速，还需要同时阅读**Cloud Data Transfer服务协议**，并选中**我已阅读并同意**以开通CDT（跨境）服务。
        
    3.  在弹出的对话框，单击**确定**。
        
    
    当**传输网络质量类型**开关右侧显示已开启联通跨境专线，表示可使用联通跨境专线加速进行访问加速。
    
3.  添加加速区域及加速IP。
    
    具体操作，请参见[添加加速区域](https://help.aliyun.com/zh/ga/user-guide/add-and-manage-acceleration-areas-1#section-fba-71z-oii)。
    
4.  添加终端节点组和终端节点，并将加速IP与终端节点组进行绑定。
    
    具体操作，请参见[添加和管理终端节点组与终端节点](https://help.aliyun.com/zh/ga/user-guide/add-and-manage-endpoint-groups-and-endpoints-for-a-basic-ga-instance#task-2253915)。
