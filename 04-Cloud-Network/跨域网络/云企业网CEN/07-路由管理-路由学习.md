通过路由学习功能，转发路由器能自动从网络实例中学习路由条目，从而减少手动配置错误的风险，提升配置效率。

## 使用限制

仅企业版转发路由器下的路由表支持创建路由学习关系。如何查看转发路由器版本，请参见[查看转发路由器实例版本](https://help.aliyun.com/zh/cen/user-guide/transit-routers#section-3qx-d5r-25n)。

## **场景示例**

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8403547771/CAEQWhiBgMDFosPyuhkiIDZlMjFhOThhOTczMjQ5M2ZiNDZiMGNhYzA0Y2I4YWFk4990858_20250320164649.277.svg)

如上图所示，假设您已在华东1（杭州）地域创建了2个VPC，且都已经连接至同1个转发路由器。

通过路由学习功能，转发路由器会将2个VPC实例的系统路由条目，学习到自己的默认路由表中。

如果2个VPC连接已[关联](https://help.aliyun.com/zh/cen/user-guide/associated-forwarding)至转发路由器的默认路由表，那么从VPC进入转发路由器的流量，会根据转发路由器自动学习的路由条目，将流量转发至对应的下一跳。

## **网络实例路由传播说明**

一个企业版转发路由器路由表支持与多个网络实例连接建立路由学习关系。一个网络实例连接也支持与多个企业版转发路由器路由表建立路由学习关系。

企业版转发路由器路由表与网络实例连接建立路由学习关系后，网络实例默认会向已建立路由学习关系的企业版转发路由器路由表传播以下路由：

| **网络实例连接** | **对应的网络实例** | **自动传播的路由** |
| --- | --- | --- |
| VPC连接 | VPC实例 | VPC实例的系统路由。 - 如果您需要将VPC系统路由表下的自定义路由传播至企业版转发路由器路由表下，请将该路由发布至云企业网中。具体操作，请参见[发布路由至转发路由器](https://help.aliyun.com/zh/cen/user-guide/advertise-routes-to-a-transit-router)。 - VPC实例自定义路由表下的自定义路由不支持发布至云企业网。 |
| VBR连接 | VBR实例 | - VBR实例**自定义路由条目**页签下一跳指向物理专线接口的路由。 - VBR实例**BGP路由条目**页签的路由。 如何查找以上路由，请参见[添加和管理路由条目](https://help.aliyun.com/zh/express-connect/user-guide/add-and-manage-routes)。 |
| ECR连接 | ECR实例 | 下一跳是VBR的路由。 |
| VPN连接 | IPsec连接实例 | IPsec连接实例目的路由表和BGP路由表中的路由。 |
| 跨地域连接 | 转发路由器实例 | - 如果企业版转发路由器连接的对端转发路由器为企业版转发路由器，且跨地域连接已开启[路由同步](https://help.aliyun.com/zh/cen/user-guide/route-synchronization)，则系统会自动将对端转发路由器下与跨地域连接建立关联转发关系的路由表下的路由传播至当前企业版转发路由器的路由表中。 - 如果企业版转发路由器连接的对端转发路由器为基础版转发路由器，且跨地域连接已开启[路由同步](https://help.aliyun.com/zh/cen/user-guide/route-synchronization)，则系统会自动将对端转发路由器路由表下的路由传播至当前企业版转发路由器的路由表中。 |

## 创建路由学习关系

### **方式一：创建网络实例连接时，直接创建路由学习**

创建网络实例连接时，在**高级配置**处选中**自动传播系统路由至转发路由器的默认路由表**，则会直接创建路由学习关系，即转发路由器默认路由表可以自动从网络实例学习路由。

以创建VPC连接为例，如图：

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3458966471/p939582.png)

详细操作，请参见[创建VPC连接](https://help.aliyun.com/zh/cen/user-guide/connect-vpcs/)、[创建ECR连接](https://help.aliyun.com/zh/cen/user-guide/connect-ecrs)、[创建VBR连接](https://help.aliyun.com/zh/cen/user-guide/connect-a-vbr-to-a-transit-router)、[创建VPN连接](https://help.aliyun.com/zh/cen/user-guide/attach-an-ipsec-vpn-connection-to-a-transit-router)、[创建跨地域连接](https://help.aliyun.com/zh/cen/user-guide/manage-inter-region-connections#18528e2013275)。

### **方式二：单独创建路由学习关系**

当遇到如下2种情况时，您可以单独创建路由学习关系。

-   创建网络实例连接时，未选中“自动传播系统路由至转发路由器的默认路由表”。
    
-   转发路由器中创建了自定义路由表，需要让自定义路由表从网络实例中学习路由条目。
    

操作步骤如下：

1.  登录[云企业网管理控制台](https://cen.console.aliyun.com/)。
    
2.  在**云企业网实例**页面，找到目标云企业网实例，单击目标实例ID。
    
3.  在**基本信息** > **转发路由器**页签，找到目标地域的转发路由器实例，单击目标实例ID。
    
4.  在转发路由器实例详情页面，单击**转发路由器路由表**页签。
    
5.  在页签左侧区域，单击目标路由表ID。在路由表详情页面，单击**路由学习**页签。
    
6.  在**路由学习**页签下，单击**创建路由学习**。在**添加路由学习**对话框，配置以下信息，然后单击**确定**。
    
    | **配置项** | **说明** |
    | --- | --- |
    | **路由表** | 系统默认选择当前路由表。 |
    | **所属转发路由器** | 系统默认选择当前转发路由器实例。 |
    | **关联连接** | 选择要与该路由表建立路由学习关系的网络实例连接。 建立路由学习关系后，网络实例会自动向路由表中传播路由，您可以在**路由条目**页签下查看路由详情。 |
    

## 删除路由学习关系

删除路由学习关系后，网络实例向企业版转发路由器路由表传播的路由会被自动撤销。

1.  登录[云企业网管理控制台](https://cen.console.aliyun.com/)。
    
2.  在**云企业网实例**页面，找到目标云企业网实例，单击目标实例ID。
    
3.  在**基本信息** > **转发路由器**页签，找到目标地域的转发路由器实例，单击目标实例ID。
    
4.  在转发路由器实例详情页面，单击**转发路由器路由表**页签。
    
5.  在页签左侧区域，单击目标路由表ID。在路由表详情页面，单击**路由学习**页签。
    
6.  在**路由学习**页签下，找到目标网络实例连接，在**操作**列单击**删除**。
    
7.  在**删除路由学习**对话框，确认信息，然后单击**确认**。
