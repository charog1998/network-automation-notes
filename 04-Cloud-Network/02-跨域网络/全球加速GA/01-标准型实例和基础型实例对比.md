全球加速 GA（Global Accelerator）包含标准型和基础型。本文为您介绍标准型全球加速 GA（Global Accelerator）实例和基础型GA实例的主要功能区别与计费差异，便于您按需选择。

## **功能特性对比**


| **功能项** | **标准型GA实例** | **基础型GA实例** |
| :--- | :--- | :--- |
| **产品定位** | • 提供多接入地域和多源站地域的全连接网络，端到端加速用户访问后端服务。<br>• 四层（TCP和UDP）和七层（HTTP和HTTPS）协议加速。 | • 提供接入地域到源站地域一对一的加速网络，为用户提供点到点的加速体验。<br>• 三层（IP）协议加速。 |
| **就近接入** | **自定义就近接入**（EIP类型加速IP）：<br>• 支持自定义加速地域，通过加速IP（各加速地域独立提供）或CNAME就近接入，路径优化自主可控。<br>• 支持选择加速IP协议，可实现IPv6转换。<br><br>**自动就近接入**（Anycast EIP类型加速IP）：<br>• 全球多地域提供统一的IP入口，各地域用户通过Anycast EIP或CNAME从就近接入点接入，运维简单。 | **自定义接入**：<br>• 通过EIP类型的加速IP自定义接入。 |
| **转发能力** | **智能路由模式**：<br>• 根据延时因素自动选择就近且健康的终端节点组转发流量。<br>• 支持TCP、UDP、HTTP和HTTPS协议，支持HTTP/3接入。<br>• 支持丰富的转发规则（标头改写、重定向、重写等）。<br>• 支持多地域负载均衡、高可用容灾和健康检查。<br><br>**自定义路由模式**：<br>• 根据IP地址及端口映射表确定性转发。<br>• 支持TCP、UDP协议。<br><br>**协议优化**：<br>• 支持TCP协议就近终结。<br>• 支持SSL证书就近卸载。 | 根据您指定的加速IP与终端节点间的映射关系，将流量分配到后端服务。 |
| **后端服务类型** | **阿里云上**：ECS、ALB、NLB、CLB、OSS、ENI、自定义私网IP、阿里云公网IP、交换机（vSwitch）<br><br>**非阿里云上**：自定义公网IP、自定义域名 | **仅支持阿里云上**：<br>ENI、ECS、CLB、NLB |
| **安全管理** | • 访问日志<br>• 访问控制黑白名单<br>• TLS加密算法套件选择<br>• DDoS基础防护 | DDoS基础防护 |
| **运维能力** | • 源站探测<br>• 实例监控<br>• 实例诊断 | 实例监控 |
| **典型应用场景** | 主要面向四层或七层应用加速，例如：支付系统、企业应用加速、视频直播、游戏加速。 | 自定义应用加速，例如：使用私有协议的应用。 |
